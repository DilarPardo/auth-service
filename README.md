# Auth Service - Microservicio de Autenticación

Este es el microservicio encargado de la gestión de usuarios, autenticación y emisión de tokens para el sistema de Control de Fabricación. Desarrollado con **Laravel 11** y **Laravel Sanctum**.

## 🛠️ Tecnologías Utilizadas

*   **Framework:** Laravel 11
*   **Lenguaje:** PHP 8.1+
*   **Autenticación:** Laravel Sanctum (Bearer Tokens)
*   **Base de Datos:** MySQL

## 🚀 Requisitos Previos

*   PHP >= 8.1
*   Composer
*   Servidor MySQL (XAMPP/Laragon)
